Dojo es una biblioteca JavaScript modular de código abierto (o más específicamente un kit de herramientas JavaScript) diseñada para facilitar el rápido desarrollo de aplicaciones y sitios web multiplataforma, basados en JavaScript/Ajax.

Es un kit de herramientas de JavaScript que ayuda a ahorrar tiempo y escala en el proceso de desarrollo. Al mismo tiempo, proporciona todo lo que se necesita para crear una aplicación web. Por ejemplo, utilidades de lenguaje, componentes de interfaz de usuario y mucho más, todo en un solo lugar y diseñado para funcionar en conjunto a la perfección.
